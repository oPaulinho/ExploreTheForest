# 🌲 Forest Explore (DEMO)

**PT-BR:** Um jogo de sobrevivência e coleta desenvolvido com **Python 3.12** e **Pygame 2.6**.
**EN:** A survival and collection game developed with **Python 3.12** and **Pygame 2.6**.

---

## 🚀 Como Jogar / How to Play

1.  **Certifique-se de ter o Python instalado.** / **Make sure you have Python installed.**
2.  **Instale o Pygame / Install Pygame:** `pip install pygame`
3.  **Instale o Pillow (para ativos) / Install Pillow (for assets):** `pip install Pillow`
4.  **Execute o jogo / Run the game:** `python main.py`

## 🕹️ Funcionalidades / Features

- **3 Modos de Jogo / 3 Game Modes:** Solo, Co-op e Competitivo.
- **3 Dificuldades / 3 Difficulties:** Easy, Medium e Hard.
- **Ciclo de Fase / Level Cycle:** Manhã (Fase 1) e Noite (Fase 2 - Multiplicador x2) / Morning (Level 1) and Night (Level 2 - x2 Multiplier).
- **Ranking:** Sistema de Top 10 Scores salvos localmente / Top 10 Scores system saved locally.

## 👥 Controles / Controls

### Player 1 (Guerreiro / Swordsman)
- **Movimento / Movement:** Setas do Teclado / Arrow Keys (↑ ↓ ← →)
- **Corrida / Run:** 'Ctrl Direito' / 'Right Ctrl'

### Player 2 (Arqueira / Archer)
- **Movimento / Movement:** `W`, `A`, `S`, `D`
- **Corrida / Run:** 'Ctrl Esquerdo' / 'Left Ctrl'
