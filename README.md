# 🌲 Forest Explore (Beta)

Um jogo de sobrevivência e coleta desenvolvido com **Python 3.12** e **Pygame 2.6**. 

## 🚀 Como Jogar

1.  Certifique-se de ter o Python instalado.
2.  Instale o Pygame: `pip install pygame`
3.  Instale o Pillow (para ativos): `pip install Pillow`
4.  Execute o jogo: `python main.py`

## 🕹️ Funcionalidades

- **3 Modos de Jogo:** Solo, Co-op e Competitivo.
- **3 Dificuldades:** Easy, Medium e Hard (selecionáveis no menu).
- **Ciclo de Fase:** Manhã (Fase 1) e Noite (Fase 2 - Multiplicador x2).
- **Ranking:** Sistema de Top 10 Scores salvos localmente.

## 👥 Controles

### Player 1 (Guerreiro)
- **Movimento:** Setas do Teclado (↑ ↓ ← →)
- **Corrida:** Teclado Numérico `ENTER`

### Player 2 (Arqueira)
- **Movimento:** `W`, `A`, `S`, `D`
- **Corrida:** `L` Shift
